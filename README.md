# Aprendizado JavaScript

Repositório criado para armazenar exercícios, exemplos e anotações durante o aprendizado da linguagem JavaScript.

## Conteúdos estudados

- Variáveis
- Tipos de dados
- Operadores aritméticos
- Operadores de comparação
- Operadores lógicos
- Estruturas condicionais (`if`, `else`)
- Laços de repetição
- Manipulação de strings
- Métodos como `trim()` e `substring()`
- Arrays (listas)
- Atribuição de valores
- Exercícios práticos

## Estrutura

Cada arquivo representa um assunto ou exercício específico, facilitando a organização e o estudo dos conceitos.

Exemplos:

- `primeiro.js`
- `operadores.js`
- `comparacao.js`
- `logico.js`
- `match.js`
- `lista.js`
- `trim.js`

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/ricardopjunkes/Aprendizado_Java_Script.git
```

2. Acesse a pasta:

```bash
cd Aprendizado_Java_Script
```

3. Execute qualquer arquivo JavaScript utilizando o Node.js:

```bash
node nome-do-arquivo.js
```

Exemplo:

```bash
node trim.js
```

## Tecnologias

- JavaScript (ES6+)
- Node.js
- Visual Studio Code

## Objetivo

Este repositório tem como objetivo documentar minha evolução no aprendizado de JavaScript por meio de exercícios práticos, testes e exemplos de código, servindo como material de consulta e revisão.
